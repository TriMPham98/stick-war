# Stick War — Project Rules

> This project uses `.claude/CLAUDE.md` as the primary rules file (newer convention). This file exists for broad compatibility.

See [.claude/CLAUDE.md](.claude/CLAUDE.md) for the full project guidelines, architecture preferences, coding conventions, and current focus areas.

Key highlights:
- Faithful but modern Three.js recreation of the 2009 Stick War.
- Plain-data simulation + thin rendering sync.
- Fixed 30 Hz timestep.
- Asset-free procedural style.
- Prioritize fun core loop and iteration speed.

When making changes, read `.claude/CLAUDE.md` first.
